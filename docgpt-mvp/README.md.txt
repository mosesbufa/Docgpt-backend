# DocGPT – MVP Health Assistant

An MVP version of DocGPT – an AI-powered personal health assistant that tracks health data and gives insights using GPT-4.

## ✅ Features

- User Onboarding
- Daily Logging (sleep, steps, mood, meals, symptoms)
- GPT Analysis & Suggestions
- Weekly Summary with AI advice
- Recharts Visualizations
- LocalStorage Reminders

## 🚀 How to Run Locally

1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Visit http://localhost:3000

## 🌐 Deployment

Deploy to Vercel via GitHub integration.
